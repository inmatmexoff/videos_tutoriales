# **App Name**: HolaMundo API

## Core Features:

- API Endpoint: Serves a JSON response with the message 'Hola mundo' when accessed via a GET request.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calm and approachable feel.
- Background color: Very light blue (#F0F8FF), almost white, creating a clean look.
- Accent color: Muted green (#8FBC8F) to add a subtle contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
